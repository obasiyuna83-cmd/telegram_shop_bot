# 🛍 Telegram E-Commerce & Service Bot

Профессиональный Telegram-бот для продажи товаров и услуг с интеграцией асинхронной базы данных. Проект разработан на базе современного фреймворка **aiogram 3.x** и асинхронной СУБД **SQLite**. 

Данный проект отлично подходит в качестве демонстрационного примера (портфолио) фриланс-разработчика, показывая навыки проектирования архитектуры, работы с API Telegram, асинхронного программирования на Python и работы с базами данных.

---

## 🚀 Основные возможности

- **🛍 Интерактивный каталог товаров:** Динамическая генерация инлайн-кнопок на основе базы данных.
- **📄 Детальная карточка товара:** Просмотр описания, цены и кнопки покупки для каждого товара отдельно.
- **💳 Покупка в один клик:** Эмуляция оформления заказа с записью в базу данных.
- **🗄 Асинхронная база данных (SQLite):** Таблицы пользователей, товаров и заказов. База данных инициализируется автоматически при первом запуске.
- **⚙️ Современный стек:** Полностью асинхронный код на Python 3.10+ с использованием `aiogram 3` и `aiosqlite`.

---

## 🛠 Технологический стек

* **Язык программирования:** Python 3.10+
* **Библиотека Telegram API:** [aiogram 3.x](https://github.com/aiogram/aiogram) (Асинхронная библиотека нового поколения)
* **Работа с базой данных:** [aiosqlite](https://github.com/omnilib/aiosqlite) (Асинхронный драйвер для SQLite)
* **Логирование:** Стандартный модуль `logging`

## 💻 Установка и запуск

### 1. Клонирование репозитория (или создание папки)
Перейдите в папку с проектом:
```bash
cd telegram_shop_bot
```

### 2. Установка зависимостей
Рекомендуется использовать виртуальное окружение (`venv`):
```bash
python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для Linux/macOS:
source venv/bin/activate

pip install -r requirements.txt
```

### 3. Настройка бота
1. Перейдите в Telegram и найдите официального бота [@BotFather](https://t.me/BotFather).
2. Создайте нового бота с помощью команды `/newbot`, задайте ему имя и юзернейм.
3. Скопируйте полученный **API Token**.
4. Откройте файл `bot.py` и вставьте ваш токен в переменную `BOT_TOKEN` на строке 11:
   ```python
   BOT_TOKEN = "ВАШ_ТОКЕН_ЗДЕСЬ"
   ```
5. *(Опционально)* На строке 181 укажите ваш Telegram юзернейм в поддержке:
   ```python
   "✍ Напишите мне в личные сообщения: @ВАШ_ЮЗЕРНЕЙМ"
   ```

### 4. Запуск бота
Запустите скрипт:
```bash
python bot.py
```
При первом запуске бот автоматически создаст базу данных `shop.db`, наполнит её тремя тестовыми товарами и выведет в консоль сообщение: `Бот успешно запущен и ожидает сообщений...`
